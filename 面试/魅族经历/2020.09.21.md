## npm 包依赖管理

在生产环境中使用 npm ci 代替 npm i

- package-lock.json 必须存在，否则 npm ci 会报错

- 自动删除 node_modules

- 不能单独装包 只能用来安装整个项目的依赖

* npm ci 比 npm install 更快

由于它不使用缓存并安装确切的软件包版本，npm ci 可以比 npm install
